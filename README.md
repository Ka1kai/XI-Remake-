# XI-Remake-

## 実行環境

Windows 7, 8, 10 動作確認済  
Microsoft DirectX SDK (June 2010)


## 操作

|画面|操作|Xbox Controller|KeyBoard|
|:---:|:---:|:---:|:---:|
|タイトル|決定|STARTボタン|SPACE|
|モードセレクト|選択 |Aボタン|SPACE|
|ハイスコア|戻る|Bボタン|SPACE|
|ゲーム|移動|十字ボタン|ARROW| 
|ゲーム|ポーズ|STARTボタン|ESCAPE| 
|リザルト|進む|Aボタン|ENTER|
